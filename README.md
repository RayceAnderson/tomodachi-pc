## Overview
**Tomodachi PC** brings the full Nintendo Switch version of Tomodachi Life to Windows using the Ryujinx emulator with optimizations for better performance and visuals.

Create Miis, enjoy chaotic island life, and experience the game with enhanced resolution and smoother gameplay.

> [!Important]
> Intended for users who legally own the game.

## Key Features
- Full game support (Ryujinx / Ryubing)
- 60FPS & High Resolution support
- Working Mii Creator & Editor
- Save file management
- Graphics and performance improvements
- Mod support
- Easy setup and regular updates
- Optimized emulator configuration

## How to Use
1. Download the latest release
2. Extract the archive
3. Install Ryujinx (recommended version included or linked)
4. Add the game files
5. Run the game and enjoy!
